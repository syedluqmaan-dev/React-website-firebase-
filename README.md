## 🚀 Features

- 🖥️ Fully Responsive on all screen sizes
- ⚡ Clean & modern UI design
- 📄 Multi-page layout (About, Services, Contact, Blog, Careers, etc.)
- 📞 Contact form & FAQs section
- 🔐 Privacy Policy & Terms Pages
- 🌐 SEO-friendly and fast-loading


## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
- Firebase Hosting
